## Topic Evaluation Criteria

Before adding a topic to the queue, evaluate against these criteria:

### Required (Must Pass All)
- [ ] **Audience Relevance:** Does the ICP (Mid-Market Squeezed) care about this?
- [ ] **Business Alignment:** Does it connect to the AI Readiness & Adoption offer?
- [ ] **Voice Fit:** Can it be written in Alvis House voice authentically?

### Scoring (Rate 1-5)
| Criterion | Score | Notes |
|-----------|-------|-------|
| **Timeliness:** How urgent is this topic? | /5 | |
| **Evergreen Potential:** Can this be repurposed later? | /5 | |
| **Differentiation:** Is our take unique? | /5 | |
| **Research Depth:** Do we have enough material? | /5 | |
| **Extraction Potential:** Can this produce derivative content? | /5 | |

**Minimum Score to Add:** 15/25
