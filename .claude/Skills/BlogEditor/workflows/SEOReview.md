# SEO Review Workflow

**Focused SEO optimization for blog content.**

Use this workflow when you need to optimize content for search without full editorial review.

---

## Purpose

Optimize blog posts for:
- Search engine visibility
- Keyword targeting
- On-page SEO elements
- Readability for SEO
- Technical SEO factors

**Time investment:** Moderate, focused on search optimization.

---

## When to Use

✅ **Use SEO Review when:**
- User asks for "SEO optimization"
- Content is editorially solid but needs search optimization
- Targeting specific keywords
- Want to improve organic traffic potential
- Competing for specific search terms

❌ **Don't use SEO Review for:**
- First comprehensive edit (use FullEdit, which includes SEO)
- Content quality issues (use FullEdit first)
- Voice or brand consistency (use VoiceCheck)

---

## 🚨 SEO REVIEW STEPS

```
INPUT: Blog draft + target keywords (optional)
     ↓
[1] RESEARCH: Keyword analysis (if not provided)
     ↓
[2] AUDIT: On-page SEO elements
     ↓
[3] ASSESS: Readability for SEO
     ↓
[4] RECOMMEND: Optimization actions
     ↓
OUTPUT: SEO scorecard + specific improvements
```

## Step 1: Keyword Research

### If user provides keywords:
- Primary keyword: [provided]
- Secondary keywords: [provided]
- Skip research, proceed to audit

### If no keywords provided:
**Identify from content:**
1. What's the main topic/concept?
2. What would someone search to find this?
3. What problem does this solve?

**Recommend primary keyword:**
- 2-4 words
- Search intent match (informational/navigational/commercial)
- Reasonable search volume (use judgment based on topic)
- Content naturally addresses this query

**Recommend 2-3 secondary keywords:**
- Related long-tail variations
- Semantic variations
- Related questions people ask

## SEO Best Practices Reference

### Keyword Usage (Natural Integration):
```
✅ GOOD:
"When you amplify humans with AI instead of replacing them,
teams achieve 10x productivity gains. Here's how human
amplification works..."

❌ BAD (Stuffing):
"Human amplification with AI amplifies humans. AI amplification
for amplifying human teams. Amplify your AI amplification..."
```

### Title Optimization:
```
✅ GOOD:
"Stop Deploying AI, Start Amplifying Humans" (45 chars, keyword, compelling)

❌ TOO LONG:
"The Complete Comprehensive Guide to Stopping AI Deployment and Starting Human Amplification" (90 chars, cut off in results)

❌ BORING:
"AI and Humans" (too generic, no hook)
```

### Meta Description:
```
✅ GOOD (158 chars):
"88% of companies deploy AI but only 5% see ROI. Learn the amplification framework that multiplies human capability instead. Start here."

❌ TOO LONG (190 chars):
"Organizations are investing heavily in AI deployment but failing to see returns. This comprehensive guide explains the complete framework for human amplification that delivers real results."

❌ TOO VAGUE:
"Learn about AI and humans working together."
```

## SEO Workflow Checklist

**Before starting:**
- [ ] Identify or confirm target keywords
- [ ] Understand search intent
- [ ] Check competitive landscape

**During audit:**
- [ ] Run on-page SEO checklist
- [ ] Assess readability scores
- [ ] Evaluate content depth
- [ ] Note all optimization opportunities

**Report generation:**
- [ ] Prioritize fixes (critical → important → nice-to-have)
- [ ] Provide specific, actionable recommendations
- [ ] Include before/after examples
- [ ] Estimate effort required

**After report:**
- [ ] If user requests, apply optimizations directly
- [ ] Recheck scores after changes
- [ ] Verify no keyword stuffing introduced

---

## When SEO Conflicts with Quality

**IMPORTANT:** Never sacrifice content quality for SEO.

**If forced to choose:**
1. **Reader value > Keyword density**
2. **Natural language > Forced keywords**
3. **Content depth > Word count targets**
4. **Brand voice > SEO templates**

**The rule:** Write for humans, optimize for search. Never the reverse.

---

**SEO is amplification for great content, not a substitute for it.**
