# Blog Editorial Frameworks

**Systematic analysis tools for editorial review.**

Use these frameworks when editing blog content to ensure consistency, quality, and reader impact.

```

SECONDARY KEYWORDS: "[keyword 2]", "[keyword 3]"

ON-PAGE SEO:
□ Primary keyword in title/H1
□ Primary keyword in first 100 words
□ Primary keyword in URL slug
□ Secondary keywords naturally integrated (not stuffed)
□ Header hierarchy logical (H1 → H2 → H3)
□ Meta description optimized (155-160 chars, includes CTA)
□ Image alt text descriptive and keyword-aware
```

## Quick Editorial Checklist

Use for fast final polish:

**STRUCTURE:**
- [ ] Compelling headline that delivers on promise
- [ ] Strong opening hook (first 2 sentences)
- [ ] Clear thesis statement (first 2 paragraphs)
- [ ] Logical section flow with descriptive subheadings
- [ ] Evidence supports claims
- [ ] Memorable conclusion with call to action

**CLARITY:**
- [ ] Jargon eliminated or explained
- [ ] Active voice (>90%)
- [ ] Short sentences (avg 15-20 words)
- [ ] Short paragraphs (2-4 sentences)
- [ ] Complex ideas broken down
- [ ] No redundancy or filler

**STYLE:**
- [ ] Grammar, spelling, punctuation correct
- [ ] Consistent tense and POV
- [ ] No clichés or tired phrases
- [ ] Strong verb choices
- [ ] Varied sentence structure
- [ ] Voice consistent throughout

**SCANNABILITY:**
- [ ] Descriptive subheadings every 3-4 paragraphs
- [ ] Bulleted/numbered lists where appropriate
- [ ] Bold/italic for emphasis (used sparingly)
- [ ] White space for visual breaks
- [ ] Key takeaways highlighted

**SEO:**
- [ ] Primary keyword in title, URL, first paragraph
- [ ] Natural keyword integration
- [ ] Meta description compelling
- [ ] Internal and external links
- [ ] Alt text on images
- [ ] Readability ≤ Grade 8

**VISUAL:**
- [ ] Featured image aligns with content
- [ ] Visual metaphors effective
- [ ] Supporting images recommended
- [ ] Image style consistent (Da Vinci/Excalidraw/Napkin)
