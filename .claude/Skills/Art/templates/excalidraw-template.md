# Excalidraw Style - General Prompt Template

## How to Use This Template

1. Replace `[CONCEPT]` with your main idea/topic
2. Replace `[MAIN ELEMENTS]` with what should be illustrated
3. Replace `[COLOR STRATEGY]` with which accent colors to use

5. Keep all the style constraints intact

## Universal Excalidraw Prompt Template

```
Digital whiteboard illustration in Excalidraw style on white background (#FFFFFF).

CONCEPT: [YOUR CONCEPT HERE - e.g., "Decision framework showing path A vs path B"]

MEDIUM & TEXTURE:
- Digital whiteboard illustration
- Clean white background (#FFFFFF)
- Hand-drawn with mouse/tablet quality
- Wiggly imperfect lines throughout
- Modern, approachable aesthetic
- Collaborative whiteboard energy

LINE QUALITY (CRITICAL - MANDATORY):
- Wiggly imperfect lines throughout (NOT straight vectors)
- Lines overshoot at corners slightly
- Variable line thickness (heavier on main elements, lighter on details)
- Circles as overlapping spirals (NOT perfect circles)
- Corners don't perfectly connect
- Hand-drawn whiteboard quality
- Deliberate jitter showing digital drawing tool
- Multiple strokes visible on shapes

MAIN ELEMENTS:
[DESCRIBE YOUR MAIN VISUAL ELEMENTS - Examples:]
- Central diagram showing [main concept]
- Left side: [element 1] with wiggly outline

FILLS & SHADING (MANDATORY):
- Hatched diagonal lines for ALL fills
- NEVER solid colors
- NEVER gradients
- Wiggly hatching lines at 45° angle
- Spacing creates value (closer lines = darker areas)
- Dense hatching for emphasis
- Lighter hatching for background elements

ARROWS & CONNECTORS:
- Wiggly arrows showing flow/direction
- Arrow heads as wiggly triangles
- Connection lines between related elements
- All imperfect and hand-drawn quality
- Lines overshoot where they connect

COLOR PALETTE (Excalidraw Standard):

  * Blue (#3B82F6) - Technical elements, positive paths, emphasis
  * Red (#DC2626) - Warnings, problems, negative states
  * Orange (#F97316) - Innovation, energy, highlights

- Strict color discipline

TEXT & LABELS:
- All-caps architect font throughout
- Professional but approachable

- Annotations with wiggly arrows pointing to elements

- Central diagram with supporting elements around it
- Left-to-right process flow
- Top-to-bottom hierarchy
- Before/after side-by-side
- 2x2 matrix/framework
- Radial layout from center
- Generous white space around all elements
- Clean hierarchy
- Asymmetric but balanced

CRITICAL EXCALIDRAW CHARACTERISTICS (MANDATORY):
- Wiggly imperfect lines throughout
- Hatched diagonal fills only (never solid)
- Digital whiteboard energy
- Circles as overlapping spirals
- Lines overshoot at corners
- Corners don't perfectly connect
- All-caps architect font
- Looks exactly like Excalidraw.com sketch
- Hand-drawn with mouse quality
- Multiple overlapping strokes visible
- Collaborative modern aesthetic
```

## Style Compliance Checklist

Before finalizing any Excalidraw prompt, verify:

- [ ] Background is white (#FFFFFF)
- [ ] All lines are wiggly/imperfect
- [ ] ALL fills use hatched diagonals (no solid colors)
- [ ] Text is all-caps architect font
- [ ] Circles are drawn as spirals
- [ ] Lines overshoot at corners
- [ ] Colors limited to 2-3 accents + black
- [ ] Composition has generous white space
- [ ] Looks like hand-drawn digital whiteboard
- [ ] Includes `--no perfect lines, --no solid fills`

## Color Strategy Guide

**Choose accent colors based on meaning:**

| Color | Use For | Examples |
|-------|---------|----------|
| **Blue** (#3B82F6) | Positive, technical, emphasis | Success paths, systems, key points |
| **Red** (#DC2626) | Negative, warnings, problems | Errors, decline, blockers |
| **Orange** (#F97316) | Innovation, energy, action | New ideas, highlights, changes |
| **Gray** (#9CA3AF) | Neutral, supporting, subtle | Background info, transitions |

**Rules:**
- Use 2-3 colors maximum (excluding black)
- Black is always primary color
- Choose colors that contrast well
- Assign colors consistently (don't switch meanings mid-diagram)

## Pro Tips

1. **Keep it simple** - Excalidraw style works best with clear, uncluttered diagrams
2. **Use white space** - Don't fill every pixel, let it breathe
3. **Limit text** - Labels should be concise (3-5 words max)
4. **Color discipline** - Stick to 2-3 accent colors
5. **Consistent hatching** - All fills should be diagonal hatched
6. **Wiggly everything** - No element should have perfect lines
7. **Arrow clarity** - Use arrows generously to show relationships
8. **Hierarchy** - Vary line thickness to show importance
