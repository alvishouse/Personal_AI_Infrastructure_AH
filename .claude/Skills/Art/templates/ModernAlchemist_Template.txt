## TEMPLATE USAGE INSTRUCTIONS

**How to use this template:**

1. **Replace bracketed placeholders** with your specific content:
   - [CONCEPT] → Your strategic idea in one sentence
   - [MAIN ELEMENTS] → Your primary visual components
   - [HEADING 1], [HEADING 2] → Your actual text labels
   - [Annotation text] → Your specific callouts

2. **Choose appropriate elements** from optional sections:
   - Include metaphorical objects if using business metaphors
   - Include mathematical elements if showing ratios/formulas
   - Include annotations if technical callouts are needed

3. **Specify composition details**:
   - Replace percentage placeholders (e.g., [40-60%] → 50%)
   - Choose placement options (e.g., [center / center-left] → center-left)
   - Select layout type (e.g., [quadrant / asymmetric] → quadrant)

4. **Define visual hierarchy**:
   - List what viewers should see first, second, third
   - Specify which elements get gold accent treatment
   - Describe the reading flow

5. **Maintain style consistency**:
   - Keep all "Critical Characteristics" unchanged
   - Preserve 3-color palette (cream, charcoal, gold only)
   - Maintain faint grid specification
   - Keep mono-line uniformity

6. **Add context**:
   - Explain the strategic insight being visualized
   - Describe how visual supports your message
   - Connect to broader content or campaign

**Result:** A complete, generation-ready prompt maintaining Modern Alchemist style consistency while visualizing your specific concept.
