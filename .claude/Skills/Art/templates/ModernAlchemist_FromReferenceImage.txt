This template is used when you have a reference image and want to recreate its content in Modern Alchemist / Intellectual Blueprint aesthetic.

**Usage:**
```

[Provide image path]
```

PAI will automatically:
1. Read and analyze the reference image
2. Extract content, composition, and concepts
3. Map to Modern Alchemist aesthetic using this template
4. Generate complete prompt ready for image generation

## Analysis Framework

When analyzing a reference image for Modern Alchemist conversion, systematically extract:

### 1. Core Concept
**Question:** What is this image communicating?
- Main message or insight
- Strategic framework or model
- Decision or comparison
- Process or flow

**Extract:** One sentence summarizing the concept

### 2. Primary Visual Elements
**Question:** What are the main components?
- Shapes, diagrams, frameworks
- Human figures or objects
- Charts, graphs, or visualizations
- Icons or symbols

**Extract:** List of elements to recreate

### 3. Composition & Layout
**Question:** How is it arranged?
- Center-weighted, asymmetric, grid-based?
- Main element placement and size
- Supporting elements arrangement
- White space usage

**Extract:** Spatial relationships to preserve

### 4. Text & Labels
**Question:** What text is present?
- Main title or heading
- Key labels and callouts
- Annotations or descriptions
- Data or metrics

**Extract:** All text content with hierarchy

### 5. Visual Hierarchy
**Question:** What draws the eye first?
- Primary focal point
- Secondary elements
- Supporting details
- Reading flow

**Extract:** Priority order for emphasis

### 6. Strategic Emphasis
**Question:** What's the most important element?
- Key insight or outcome
- Critical decision point
- Main message
- Call to action

**Extract:** Target for gold accent

## Style Translation Rules

### Background Conversion
| Reference Has | Modern Alchemist Uses |
|---------------|----------------------|
| White background | Warm cream (#F5F5DC) |
| Colored background | Warm cream (#F5F5DC) |
| Textured background | Cream + faint grid + subtle grain |
| Plain background | Cream + faint grid |
| Dark background | Warm cream (#F5F5DC) |

**Always:** Cream background with FAINT grid (barely visible)

### Line Style Conversion
| Reference Has | Modern Alchemist Uses |
|---------------|----------------------|
| Wiggly/hand-drawn lines | Clean uniform mono-line |
| Variable weight lines | Consistent mono-line weight |
| Sketchy lines | Precise patent illustration |
| Rough textures | Smooth digital precision |
| Multiple line styles | Single uniform line style |

**Always:** Deep charcoal (#2C2C2C) mono-line vectors

### Color Palette Conversion
| Reference Has | Modern Alchemist Uses |
|---------------|----------------------|
| Multiple colors | 3-color only (cream/charcoal/gold) |
| Bright colors | Muted gold for key emphasis only |
| Color-coded sections | Gold accent on most important only |
| Solid fills | Light grey washes if needed |
| Gradients | Minimal grey washes only |

**Always:** Cream background, charcoal primary, gold sparingly

### Typography Conversion
| Reference Has | Modern Alchemist Uses |
|---------------|----------------------|
| Sans-serif headings | Bold serif (Garamond/Baskerville) |
| Handwritten text | Clean architect serif/sans-serif |
| Mixed fonts | Two fonts max (serif + sans-serif) |
| Informal text | Professional typography |
| All-caps labels | Appropriate case with hierarchy |

**Always:** Serif headings + sans-serif body

### Texture Conversion
| Reference Has | Modern Alchemist Uses |
|---------------|----------------------|
| Parchment texture | Subtle paper grain on cream |
| Digital flat | Subtle grain + smooth precision |
| Heavy weathering | Minimal aged paper quality |
| Napkin texture | Cream with faint grid instead |
| No texture | Subtle paper grain added |

**Always:** Subtle grain background, smooth foreground

## Common Reference Image Types

### Type 1: Whiteboard/Excalidraw Style
**Reference characteristics:**
- White background, wiggly lines, hatched fills
- Informal hand-drawn aesthetic
- Multiple colors, architect font

**Modern Alchemist translation:**
- Replace white → warm cream with faint grid
- Replace wiggly lines → clean mono-line vectors
- Reduce colors → cream/charcoal/gold only
- Keep composition and concept
- Add professional sophistication

### Type 2: Da Vinci/Parchment Style
**Reference characteristics:**
- Aged parchment, cross-hatching, variable lines
- Renaissance notebook aesthetic
- Copper and slate blue inks

**Modern Alchemist translation:**
- Replace parchment → warm cream with faint grid
- Replace cross-hatching → light grey washes
- Replace variable lines → uniform mono-line
- Replace copper/blue → gold accent sparingly
- Modernize while keeping technical precision

### Type 3: Napkin Sketch Style
**Reference characteristics:**
- Napkin texture, single ballpoint pen
- Spontaneous gestural energy
- Visible corrections

**Modern Alchemist translation:**
- Replace napkin → cream with faint grid
- Replace gestural lines → precise mono-line
- Remove corrections → clean final version
- Keep concept and spontaneity of idea
- Add professional polish

### Type 4: Corporate Diagram
**Reference characteristics:**
- Clean digital, multiple bright colors
- Sans-serif fonts, gradient fills
- PowerPoint aesthetic

**Modern Alchemist translation:**
- Replace flat white → cream with faint grid
- Replace gradients → light grey washes
- Reduce colors → cream/charcoal/gold
- Replace sans-serif everywhere → serif headings + sans body
- Add vintage engineering aesthetic

### Type 5: Technical Illustration
**Reference characteristics:**
- Engineering drawing, precise lines
- Multiple line weights, dimension callouts
- Technical annotations

**Modern Alchemist translation:**
- Keep technical precision
- Unify line weights → mono-line
- Replace background → cream with faint grid
- Simplify colors → cream/charcoal/gold
- Maintain patent illustration quality
