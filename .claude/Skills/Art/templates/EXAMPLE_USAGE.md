## Key Lessons from This Example

### 1. Be Specific
Don't just say "four pillars" - specify what they represent, how they're arranged, what connects them.

### 2. Strategic Accent Usage
Gold accent on Culture foundation and Integration arch creates visual hierarchy that reinforces your message.

### 3. Reading Flow Matters
Bottom-up reading (Culture → Pillars → Integration) matches logical flow of the concept.

### 4. Metaphor Consistency
Architectural pillar metaphor throughout - structural stability = organizational readiness.

### 5. Professional Context
Modern Alchemist style + clear framework = thought leadership authority.

## Quick Template Workflow

**For any new image:**

1. **Copy template** → Start fresh
2. **Define concept** → One sentence summary
3. **List main elements** → Primary visuals
4. **Choose accent targets** → What gets gold?
5. **Write all text** → Headings, labels, annotations
6. **Specify composition** → Placement and percentages
7. **Define hierarchy** → What viewers see first
8. **Explain intent** → Strategic insight being communicated
9. **Save prompt** → Descriptive filename
10. **Generate image** → Standard command

**Time investment:** 15-20 minutes for detailed prompt = professional visual asset.

## Common Patterns

### Framework Diagrams
- Multiple components showing relationships
- Gold accent on most important element
- Clear labeling with serif headings
- Annotations explaining each component

### Process Flows
- Sequential steps left to right
- Arrows showing progression
- Gold accent on final outcome/goal
- Numbers marking sequence

### Decision Trees
- Binary or multiple paths
- Gold accent on optimal path
- Clear outcome differentiation
- Decision criteria labeled

### Comparison Diagrams
- Side-by-side contrasting approaches
- Gold accent on recommended approach
- Clear visual differentiation
- Outcomes shown with metrics

## Tips for Success

### Typography
- Use ALL CAPS for headings (authority)
- Sentence case for annotations (readability)
- Keep labels short (1-3 words)
- Annotations can be longer (full sentences)

### Gold Accent
- Less is more (5-10% maximum)
- Guide eye to key insight
- Reinforce your article's main point
- Never use on everything

### Composition
- Don't cram - use whitespace
- Align to the faint grid
- Symmetry = stability/authority
- Asymmetry = dynamic/modern

### Testing
- Generate test image first
- Review against style checklist
- Adjust accent placement if needed
- Refine text hierarchy
